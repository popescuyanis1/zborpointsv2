# Production checklist

Before every live deploy:

1. Export Firebase Realtime Database backup.
2. Confirm login with email and password works.
3. Confirm guest volunteer login works.
4. Confirm dashboard loads.
5. Confirm volunteers page loads.
6. Confirm departments page loads.
7. Confirm points legend page loads.
8. Confirm users/coordinators page loads only for managers.
9. Add a test volunteer.
10. Add test points.
11. Check that monthly totals update.
12. Check that volunteers cannot edit data.
13. Check that mobile layout works.
14. Delete the test data after testing.
15. Deploy only after testing.
